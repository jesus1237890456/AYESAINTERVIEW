0- descargar Visual C++ 2019 Redistributable de 32 o 64 bits segun el ordenador
1- instalar xampp e iniciar el servicio de mysql
2- instalar nodejs
3- instalar angular cli (npm install -g @angular/cli)
4- descargar a ser posible visual studio code o similar
5- poner vía terminal tanto de back como de front npm i --legacy-peer-deps para instalar los módulos pertienentes del proyecto
6- instalar mysql workbench e importar fichero sql ubicado en la rama backend-ayesa a nivel de raíz.
7- iniciar backend con el comando npm start e iniciar front con ng serve (puerto por defecto 4200) url de acceso (localhost:4200)
8-para descargar los repositorios, en code y descargar zip o mediante iniciación de repositorio git (agregar clave ssh previamente), adición del repositorio mediante git remote add origin git@github.com:jesus1237890456/AYESAINTERVIEW.git a continuación git pull origin rama a descargar.
